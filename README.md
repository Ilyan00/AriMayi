## Getting Started

1. Clone the repository  
2. Install all dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```
3- Start the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

To access the second page with all the candidates, add /listeCandidats to the URL.
Then, you can click on a candidate to view their information.
